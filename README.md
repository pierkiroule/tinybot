# Tiny Psybot — Qwen WebLLM

Webapp statique de chatbot utilisant WebLLM et le modèle **Qwen2.5-0.5B-Instruct-q0f32-MLC** chargé depuis Hugging Face. Tout fonctionne côté navigateur : aucun backend ni API externe.

## 1️⃣ Installer
Aucun prérequis : téléchargez ou clonez le projet et ouvrez simplement `index.html` dans un navigateur moderne (Chrome, Edge, Firefox, Safari).

## 2️⃣ Utiliser
1. Attendez le message indiquant que le modèle est chargé.
2. Saisissez votre message dans le champ en bas et cliquez sur **Envoyer**.
3. Les échanges s’affichent en bulles dans la zone centrale.

## 3️⃣ Déployer sur GitHub Pages
1. Poussez `index.html`, `style.css`, `script.js` et `README.md` sur la branche `main` de votre dépôt.
2. Dans les paramètres GitHub, activez **Pages** sur la branche `main`, dossier racine (`/`).
3. Ouvrez l’URL fournie par GitHub Pages pour utiliser le chatbot directement dans le navigateur.
